# CS2: Active Page Management & Performance Analytics

**Enterprise-grade HRM and Page Operations Platform**

CS2 เป็นระบบบริหารจัดการ Facebook Pages และ Accounts ที่เน้นการติดตามประสิทธิภาพ (Performance), การจัดการสิทธิ์ (RBAC), และการเตรียมความพร้อมสำหรับการทำ Data Analytics ในระดับสูง

## 📁 โครงสร้างโปรเจกต์ (Project Structure)

เพื่อความสะอาดของ Root Directory ข้อมูลสำคัญถูกแยกไว้ดังนี้:

- `/src`: ซอร์สโค้ดหลักของแอปพลิเคชัน (Next.js + Tailwind)
- `/docs`: เอกสารประกอบโปรเจกต์และคู่มือทางเทคนิค
  - `PROJECT_DESIGN_DOC.md`: วิสัยทัศน์และสถาปัตยกรรมข้อมูลหลัก
  - `walkthrough.md`: สรุปฟีเจอร์ที่พัฒนาแล้วและวิธีการใช้งาน
  - `PROJECT_PLAN.md`: แผนงานและ Roadmap ของโปรเจกต์
- `/legacy`: โค้ดระบบเดิมเพื่อการอ้างอิง

## 🚀 เทคโนโลยีที่ใช้ (Tech Stack)

- **Frontend**: Next.js 14 (App Router), TypeScript
- **Styling**: Tailwind CSS (Native)
- **Icons**: Lucide React
- **Database (Planned)**: Supabase
