# Walkthrough: Performance Commission & Work Submission

ระบบส่งงานและคำนวณค่าคอมมิชชันแบบ Multi-Tier ได้รับการติดตั้งเรียบร้อยแล้ว โดยแยกส่วน Logic (Services) ออกจาก UI (Components) อย่างชัดเจน

## 1. บริการคำนวณและตรวจสอบ (Services)
- [revenueService.ts](file:///c:/dev/CS2/src/services/revenueService.ts): หัวใจของการคำนวณ 3 Tiers (Penalty, Standard, Super Bonus) และการแปลง THB -> USD
- [opsService.ts](file:///c:/dev/CS2/src/services/opsService.ts): ระบบนับคลิป (40 คลิป/วัน) และตรวจสอบสถานะ "ส่งงานครบ"

## 2. การเข้าถึงหน้าส่งงาน (Dashboard CTA)
เพื่อให้พนักงานเข้าถึงหน้าส่งงานได้ง่ายที่สุด (ตามคำขอ "อยู่ใน workspace"):
- **Dashboard**: แสดงสถิติรวม (ผู้ติดตาม, ยอดการรับชม) พร้อมกราฟเส้นที่สวยงาม
- **Smart Page Insights**: เซกชันใหม่ด้านล่างกราฟ แสดงการ์ดเพจที่ **Active** พร้อมดึงรูปโปรไฟล์, คำอธิบาย และยอดผู้ติดตามจาก Facebook มาแสดงอัตโนมัติ
- **Smart Sorting**: เรียงลำดับการ์ดตาม **กล่อง (Box ID)** เพื่อให้ตรงตามโครงสร้างสายงาน
- **Direct Navigation**: การ์ดทุกใบสามารถ **คลิกเพื่อเปิดไปยังหน้า Facebook Page จริง** ได้ทันที
- **Live Sync Status**: ระบบแสดงสถานะการเชื่อมต่อ และเวลาที่ซิงค์ข้อมูลล่าสุดของแต่ละเพจ
- **Sidebar Icon**: ไอคอนรูปวิดีโอ (Video Icon) ในแถบเมนูหลัก

## 3. ปรับปรุง UI (Clean Kanban)
- **Remove "U" Logo**: ลบสัญลักษณ์ Unassigned ออกจากกล่องเพจเพื่อให้หน้าจอสะอาดและดูเป็นระเบียบมากขึ้น

## 4. หน้าส่งงานสำหรับพนักงาน (Daily Task)
สำหรับผู้ใช้ทุกบทบาท จะเห็นเมนูใหม่ "คลิปงาน (Daily)"
- **Live Session Indicator**: แสดงวันที่ปัจจุบัน (รูปแบบไทย) พร้อมจุด Pulse สีเขียว เพื่อย้ำว่าเป็นการส่งงานของวันนี้เท่านั้น (ห้ามส่งย้อนหลัง)
- **Policy Enforcement**: รองรับการกรอกลิงก์ 10 เพจ x 4 คลิป ตามกฎของ Super Admin
- **Progress Tracking**: ระบบคำนวณ Progress แบบ Real-time
- **Submission Rules**: ปุ่มบันทึกงานจะเปิดให้กดเมื่อกรอกครบ 40 คลิปเท่านั้น

## 5. ศูนย์การเรียนรู้ (Learning Center)
เพิ่มพื้นที่สำหรับพัฒนาทักษะพนักงาน:
- **Embedded Player**: ดูวิดีโอสอนงานได้ทันทีไม่ต้องสลับหน้าจอไปที่อื่น
- **Categorization**: แยกหมวดหมู่ "คอร์สบังคับ" และ "เทคนิคการทำงาน"
- **Automatic Embed Converter**: เพียงแค่วางลิงก์ YouTube หรือ Vimeo ปกติ ระบบจะเปลี่ยนเป็นโค้ดฝังให้ทันที พร้อมดึงภาพหน้าปกให้อัตโนมัติ
- **Tagging System**: สามารถสร้างหมวดย่อยหรือแท็กใหม่ได้เอง หรือเลือกจาก **Tags เดิมที่มีอยู่แล้ว** เพื่อความรวดเร็วและเป็นระเบียบ
- **Progress Tracking**: พนักงานสามารถกดปุ่ม **"Mark as Finished"** เพื่อบันทึกว่าเรียนจบแล้ว โดยระบบจะแสดงสถานะ **"Completed"** สีเขียวบนวิดีโอนั้นๆ
- **Duration Planning**: ช่องความยาวคลิป มีไว้เพื่อให้พนักงานวางแผนเวลาเรียนรู้ได้เหมาะสม (เช่น หากมีเวลา 10 นาที ก็สามารถเลือกดูคลิปที่สั้นกว่าได้)
- **Priority Sorting**: ตั้งลำดับความสำคัญ (1-10) เพื่อจำกัดลำดับการแสดงผลของวิดีโอ (Priority สูงจะขึ้นก่อน)
- **Admin CRUD**: ปุ่มแก้ไขและลบสำหรับผู้บริหาร เพื่อจัดการเนื้อหาได้แบบ Real-time

## 6. กฎระเบียบบริษัท (Policy Center)
เพิ่มศูนย์รวมเอกสารและกฎหมายบริษัท:
- **Indexed View**: แยกส่วนสารบัญที่ชัดเจน (กฎการหักเงิน, ค่าคอม, วัฒนธรรมองค์กร)
- **High-Readability UI**: ออกแบบมาให้อ่านง่าย มีการสรุปใจความสำคัญในรูปแบบการ์ด
- **Official Stamp**: แสดงสถานะการอัปเดตล่าสุดเพื่อความโปร่งใส

## 3. หน้าตั้งค่านโยบาย (Policy Settings)
สำหรับ **Super Admin** สามารถเข้าถึงผ่านแท็บ **"ระบบเงินเดือน" (Payroll)**
- ปรับเปลี่ยนเป้ายอดวิว (Min Target) และเงินหัก (Penalty)
- ปรับเรทค่าคอมมิชชันทั้ง 2 Tiers ได้ทันที
- ปรับเกณฑ์การปลดล็อค Super Bonus (เช่น 100M Views)

## 4. ปรับปรุงระบบส่วนกลาง
- **Navigation**: เพิ่มไอคอน Video และ SettingsIcon เพื่อความชัดเจน
- **Main Page**: จัดการ Role-based Rendering ให้เหมาะสมกับแต่ละตำแหน่ง
- **Type Safety**: เพิ่ม interface [DailySubmission](file:///c:/dev/CS2/src/types/index.ts#112-120) และ [PolicyConfiguration](file:///c:/dev/CS2/src/types/index.ts#121-130) เพื่อความแม่นยำของข้อมูล
